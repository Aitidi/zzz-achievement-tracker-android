# ZZZ成就

一个为《绝区零》玩家准备的离线成就追踪 App。  
轻量、无广告、数据本地保存。

---

## ✨ 功能亮点

- ✅ 成就勾选与进度统计
- 🔎 按关键词搜索成就
- 🧩 按版本 / 分类筛选
- ↕️ 按版本新旧排序
- 📊 分版本完成率统计
- 💾 本地导入 / 导出进度（JSON）
- 🛡️ 重置保护

---

## 📱 下载与安装

请前往 **Releases** 页面下载最新安装包（APK）。

> 如果提示“无法安装”，请先卸载旧版本后再安装（通常是签名差异导致）。

---

## 🚀 使用说明

1. 在“成就”页勾选已完成项目
2. 用筛选 / 搜索快速定位目标成就
3. 在“统计”页查看整体与分版本完成情况
4. 在“设置”页进行导入、导出与重置

---

## 🔐 数据与隐私

- 所有进度默认保存在本地设备
- 不上传服务器，不做账号绑定
- 你可以随时导出 JSON 备份
